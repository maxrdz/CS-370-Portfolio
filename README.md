# CS-370 | Current/Emerging Trends in CS | Portfolio

In this project, I developed a pirate intelligent agent using reinforcement learning to navigate a maze and find treasure.
I was provided with the project framework, including the environment, helper functions, and some starter code, while I
implemented the Q-learning algorithm and training logic. This project strengthened my understanding of neural networks,
intelligent agents, and reinforcement learning by applying AI concepts to solve a practical problem.
